Ruby on Rails Developer

I chose Ruby because of its beauty, very nice syntax and libraries for all occasions. Now I'm continuing to develop my skills as a Backend developer, continuing to learn design patterns and algorithms. I also recently got interested in Web3 technologies and started trying to write my own smart contracts for the Ethereum network.
